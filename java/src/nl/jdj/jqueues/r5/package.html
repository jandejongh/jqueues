A package for simulation of queueing systems.

A (far more) detailed description of this package is available in the javadoc of SimQueue in this package. In the package, we uses three important concepts:

  • Jobs (interface SimJob): a job represents an amount of work to be carried out by one or more servers
  • Queues (interface SimQueue): a queue is an abstract waiting line holding jobs to be served by one or more servers associated with the queue;
  • Entities (interface SimEntity): the common part of jobs and queues.
The so-called queue discipline determines in what order jobs are served by the server(s).

Notes:

  • This package depends on the jsimulation package.
  • This package focuses exclusively on modeling and simulation of the queueing systems themselves; it currently does not feature adequate random-number and stochastic-process generation and only basic statistics (mostly because we believe there are extremely good Java libraries for that already).
